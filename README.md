# Library Endterm Project API (Spring Boot)

## Run in IntelliJ
Run:
`src/main/java/com/example/library/Application.java`

## DB (pgAdmin)
Create DB `library_endterm` then run:
`db/schema.sql`

## Endpoints
- GET  /api/health
- GET  /api/books
- POST /api/books
- PUT  /api/books/{id}
- DELETE /api/books/{id}

Postman collection:
`postman/Library Endterm API.postman_collection.json`
